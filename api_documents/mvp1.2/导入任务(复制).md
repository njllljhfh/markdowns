# 导入任务（复制任务）

[Url]: 127.0.0.1:8000/training/tasks/duplication



## 1、前端传来的数据

- **`请求类型`**：**POST**

```json
{
    "task_id_old":2018102439652891014, # 已有的任务id
    "task_id":2018102439652891056  # 新建任务的id
}
```



## 2、返回给前端的数据

```json
# 成功
{
    "code": 0,
    "msg": "复制任务成功"
}

# 失败
{
    "code": 错误码,
    "msg": "错误信息"
}
```
