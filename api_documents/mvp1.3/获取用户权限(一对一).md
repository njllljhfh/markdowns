# 获取用户权限信息（一对一）

> 设计人：牛金龙



## Web调用

- 描述：获取用户的权限信息

- 请求URL：/api/users/info/<user_id>

- 请求类型：GET

- 请求参数

  ```pyhon
  user_id: 用户id
  ```


- 返回的数据

  ```json
  {
    "code":0,
    "msg":"获取 用户id={self.user_id} 的权限信息成功",
    "uinfo_dict":{
          		"user_id": "2018120739652891014",
          		"user_name": "xiaofeng",
          		"create_time": "None",
          		"state": "启用",
                  "org":  {	# 该用户所属的组织机构
                              "org_id":"2018102439652891014",  # 用户所属组织机构的id
                              "org_name":"研发部",  # 用户组织机构名称
      					},  
                  "group":{	# 用户组
                              "group_id":"2018102439652891014",   # 户组id
                              "group_name":"研发一组",  # 用户组名称
      					},
                                
        			"role":{	#该用户的角色
                              "role_id":"1",  # 角色id
                              "role_name":"总工程师",  # 角色名称

      					},

                      				"func_id":1,  # 功能id
  ```
