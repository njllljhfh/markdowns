# 删除用户信息

> 设计人：牛金龙



## Web调用

- 描述：删除用户信息

- 请求URL：/api/users/info

- 请求类型：DELETE

- 请求参数

  ```python
  {
      "user_id_ls":["2018120739652891111","2018120739652892222"],  # 要删除的用户id 列表
      "create_oper":"2018120739652891014"  #  当前登录的用户的id
  }
  ```

- 返回的数据

  ```json
  {
    "code":0,
    "msg":"删除用户信息 成功",
  }
  ```
